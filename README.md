# my-vue-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### Build Image
```
docker build -t vue .

```

### Create Container
```
docker run --name my-vue -itd -p 80:8080 vue
```
